## Thinking

Phase 0 of the Wick implementation plan. The directory is empty — need to create the full workspace structure so that cargo build succeeds. This is pure scaffolding with stub types and no real logic.

Key decisions:
- Edition 2024 requires Rust 1.85+
- Two crates: `wick` (lib) and `wick-cli` (bin)
- Stub all modules from the plan so the file tree is established
- Include enough type definitions to make the module structure meaningful
- Quant block structs need `repr(C, packed)` + size assertions for GGUF compatibility
